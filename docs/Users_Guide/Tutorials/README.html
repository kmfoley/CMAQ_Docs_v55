CMAQ Tutorials#

A series of short tutorials provide practical examples of how to set up and run CMAQ, exercise different features in the software, and better understand the software system. Community members are encouraged to both suggest ideas for new tutorials and to contribute tutorials directly to this repository. Contact the CMAS Center with new tutorial ideas.

  • Scripts to install software needed to run CMAQ

    • Script to install I/O API library to support netCDF4 compression

    • Script to install netCDF libraries to support netCDF4 compression

    • Script to install CMAQv55 for CB6r5 and M3DRY

  • Directions on how to prepare your Linux system for installing and running CMAQ (These directions are an alternative to the above helper scripts.)

    • Building CMAQ for GNU using libraries that disable netCDF4 compression

    • Building CMAQ for GNU using libraries that support compressed netCDF4

    • Building CMAQ for Intel using libraries that disable netCDF4 compression

    • Running the CMAQ Base Model with CB6r5 and M3DRY or STAGE

    • Running the CMAQ-ISAM Test Case

    • Running the CMAQ-DDM-3D Test Case

    • Running the WRF-CMAQ Test Case
